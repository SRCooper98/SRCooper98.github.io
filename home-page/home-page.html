FM Scouting Tool

How to use

To use the tool, export a html file from FM using the views linked which are views which will show all the attributes for the players. This includes views for your squad, player search and shortlist
Once you have exported a html file you can upload it here. It will ask you to save the output, save this as a html file.
It will give you a file that is in a table format of all the players and allow you to sort each column easily
