What's this tool for?

This tool will help analyse and discover talent when scouting in Football Manager. It takes in all the attributes of the players and rates them 1 to 20 on all roles in the game for you to assess. It can reveal players who can play in a position not though possible or it allows you to assess the quality of the teams around you to compare and see where the weaknesses are in your own team

How to use:

1.

Apply the appropriate view to the selected screen on FM

Squad view applied to squad screen etc...

2.

The views will display all player attributes

Must have full scouting of player or have attribute masking off

3.

Export the screen as a html document. Limited to 250 Players

1. Select the players to output or leave blank to grab all
2. Ctrl + P
3. Export as Web Page

4.

Upload HTML file here

5.

Save the file as a html file wherever you want

Open file with any web page to view. You can also sort columns for quick check of best in postion
